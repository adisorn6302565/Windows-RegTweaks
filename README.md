# Windows Gaming Optimization Registry Tweaks

## คำอธิบายโปรเจกต์
โปรเจกต์นี้รวบรวมการปรับแต่ง Windows Registry และสคริปต์เพื่อเพิ่มประสิทธิภาพของระบบสำหรับการเล่นเกม โดยแบ่งออกเป็น 10 หมวดหมู่หลัก พร้อมไฟล์ .reg และ .bat สำหรับการใช้งาน

## วิธีการใช้งาน

### ขั้นตอนที่ 1: สำรองข้อมูล
ก่อนใช้งาน ให้สำรอง Registry และสร้างจุดคืนค่า (System Restore Point):
- คลิกขวาที่ This PC > Properties > System Protection > Create
- หรือใช้คำสั่ง: `reg export HKLM system_backup.reg`

### ขั้นตอนที่ 2: เลือกการปรับแต่ง
- เลือกหมวดหมู่ที่ต้องการปรับแต่งจากโฟลเดอร์ย่อย
- เปิดไฟล์ .reg ที่ต้องการ (ดับเบิลคลิก) และยืนยันการนำเข้า
- หรือใช้ไฟล์ `game_optimization.bat` เพื่อปรับแต่งทั้งหมด (รันด้วยสิทธิ์ Administrator)

### ขั้นตอนที่ 3: รีบูตระบบ
หลังจากปรับแต่ง ให้รีบูตเครื่องเพื่อให้การเปลี่ยนแปลงมีผล

### ขั้นตอนที่ 4: เล่นเกม
หลังเล่นเกมเสร็จ ให้เปิดใช้งานฟีเจอร์ความปลอดภัยที่ปิดไว้ใหม่ (เช่น Windows Defender)

## คำเตือน
- **ความเสี่ยงด้านความปลอดภัย**: การปรับแต่งบางอย่างอาจลดความปลอดภัยของระบบ (เช่น ปิด Windows Defender, Spectre/Meltdown Mitigations)
- **ความเสถียรของระบบ**: การเปลี่ยน Registry อาจทำให้ระบบไม่เสถียร หากเกิดปัญหา ให้ใช้ System Restore
- **ไม่รับประกัน**: ผู้ใช้ใช้ความเสี่ยงเอง ควรทดสอบในสภาพแวดล้อมที่ปลอดภัย
- **เฉพาะ Windows**: สคริปต์นี้สำหรับ Windows เท่านั้น

## การเขียนและพัฒนา
โปรเจกต์นี้พัฒนาโดยใช้ความรู้ด้าน Windows Registry และการปรับแต่งระบบสำหรับการเล่นเกม โดยรวบรวมการตั้งค่าที่เป็นที่นิยมและปลอดภัยที่สุดเท่าที่จะทำได้ แต่ละไฟล์ .reg มีคำอธิบายและคำเตือนภายใน
